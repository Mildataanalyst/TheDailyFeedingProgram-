# v78 — Admin Undo / Redo UI

Adds an Admin Undo / Redo panel in the places where mistakes are most likely:

1. Lead Pool — undo lead imports, curation decisions, deletes, and sending approved leads to PM shortlisting.
2. PM Shortlists / Combined Review — undo PM task/response changes that affect the combined board.
3. Final Ranking / Contact Tracker — undo sending final buckets into the tracker and tracker edits/removals.

The panel requires the same `ADMIN_PASSWORD` that is already configured on the backend.
