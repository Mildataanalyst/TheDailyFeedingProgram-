# V90 — Premium white landing theme

This release keeps the V87 landing page structure, copy, Lexend typography, spacing, card geometry, hover behaviour and reveal animations unchanged.

Only the landing-page visual palette was translated from black to a premium warm-white system:

- warm white / paper backgrounds
- black primary typography
- Feeding India red accents and CTAs
- white editorial cards with restrained shadows
- subtle red ambient glow
- slightly increased spacing between `2` and `.` in `2.0`

No routes, links, workflow logic, preview behaviour or application pages were changed.
