# Frontend v150 — Guest gearbox and seamless Guest scoring

- Adds a dedicated Guest gearbox inside the Guest workspace.
- Admin can choose a Program Manager, search/select a specific NGO, and duplicate it into Guest.
- Duplication uses the existing blind-copy backend flow: source assignment, PM progress, response, and official ranking remain unchanged.
- Prevents accidental duplicate copies of the same source task.
- Opens the newly copied NGO immediately in the Guest scoring workspace.
- Guest now has the same Reference examples and Scoring tutorial controls as PM scorers.
- Guest empty state directly opens the NGO picker.
