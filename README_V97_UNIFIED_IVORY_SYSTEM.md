# V97 — Unified Ivory System

This release applies the landing page's warm ivory / black / Feeding India red design language across the operational frontend.

## Landing polish
- Removed the FI background hero object.
- Kept the hero copy limited to `DFP 2.0` and `Find the best NGOs`.
- Reduced and optically re-centred the heart decimal between `2` and `0`.
- Refined the FI + Scroll down cue and its downward motion.

## Whole-site visual system
- Warm ivory canvas with low-opacity tactile grain.
- Translucent acrylic surfaces, fine white borders, soft inner highlights, and restrained shadows.
- Unified glass navigation with clear red active state.
- Consistent page heroes, cards, inputs, uploads, tables, filters, status panels, trackers, PM workspaces, rankings, modals, and drawers.
- Feeding India red retained for primary actions, focus states, and meaningful emphasis.
- Responsive and reduced-motion behaviour preserved.

## Scope
- Landing page
- NGO Discovery
- Repository
- Story Discovery
- Rankings / PM workstream
- Contact Tracker
- Shared headers, forms, tables, drawers, and modals

The preview page itself was intentionally left for a later dedicated pass.
