# v81 — NGO Presence Check UI

Adds NGO Presence Check inside Internet Leads → Advanced settings.

The UI supports:

- Upload CSV with `ngo_name,state,center_name`
- Sample CSV download
- Run/cancel controls
- Live status showing processed NGOs, output rows, and Serper queries used
- Downloads for Presence CSV, Summary, and Audit
- Small live preview table
- Archive/history support for previous Presence Check runs

Presence Check runs are not shown with the Lead Pool import button by default, because the output is meant for web-presence assessment rather than automatic lead approval.
