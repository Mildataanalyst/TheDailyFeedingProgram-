# V119 — Single exception override

The three metric cards now contain only:

- a 1–5 metric score;
- a minimum 100-character rationale; and
- the evidence pack and recommended ceiling.

There is no per-metric override.

After all three cards, one optional **Exception override** can be opened. It stores:

- `enabled`
- `rank` from 1–5
- `reason` with a 100-character minimum

This is an overall exception judgement for the NGO. It does not alter the three metric scores or the locked earlier overall ranking.

A metric score above its evidence-pack ceiling is blocked. The PM must keep that metric at or below the ceiling and use the single exception override when the overall NGO judgement needs to differ.
