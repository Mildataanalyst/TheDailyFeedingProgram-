# DFP 2.0 V111 — Premium Depth System

This build applies one premium visual system across the complete NGO Discovery and Rankings workflow, including PM Shortlists, Combined Review, and Final Ranking.

## Main visual changes
- One solid neutral canvas and one exact white operational surface.
- Landing-page-scale typography on internal page mastheads and hubs.
- Tactile elevated cards using borders, inset highlights, and restrained 3D shadows rather than competing colors.
- Consistent raised red primary buttons and crisp raised white secondary buttons.
- Wider desktop workspace and stronger responsive sizing.
- Premium PM Shortlists dashboard and reviewer workspace.
- Elevated reviewer summary tiles and rating bands in Combined Review.
- Showcase decision-room hierarchy in Final Ranking.
- Champagne-gold animated In-depth control with light sweep and sparkle.
- Explicit dark-text-on-white contrast lock throughout operational surfaces.
