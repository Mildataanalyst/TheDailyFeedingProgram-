# Frontend v148 — Reference after scoring

- The PM calibration/reference panel is hidden while the reviewer chooses a practice score.
- Selecting 1–5 records the reviewer choice with a tick and then reveals the full PM reference rankings.
- The selected score remains visible and locked while the reviewer compares it against the reference.
- The reference screen remains open until the reviewer clicks Continue; it no longer auto-advances after less than a second.
- No correctness or affirmation message is shown and no reference row is highlighted.
