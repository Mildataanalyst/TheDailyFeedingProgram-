# V144 — Guest and Tanishq shortlisting

- Adds a Guest card branded only with the official Feeding India by Eternal Foundation logo, the Guest name and a Shortlist action.
- Admins can copy any PM shortlist assignment into Guest. The source PM assignment, progress and official score stay unchanged.
- Guest receives the NGO information and the same three-metric scoring flow, but no earlier PM score is shown before submission.
- After Guest submits all three metric scores, a Thank You modal reveals the source DFP PM's ranking.
- Guest assessments remain outside the official combined PM score and PM-progress leaderboard.
- Tanishq now uses the same three-metric shortlisting workspace and card architecture as every other PM.
- Removes the old NGO-details/referral-only Tanishq workflow from the frontend.
